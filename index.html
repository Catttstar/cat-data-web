中央大學歷年交換學生申請數量

[ 背景簡介 ]

交換學生申請地區與國際局勢、地緣政治和個人偏好有關。 中大國際處提供之各學年申請第一志願地區數量紀錄，將資料繪製成折線圖可觀察近年來學生的出國選擇偏好變化。 可觀察申請總數在110學年度的申請總人數驟降，很可能是因為COVID-19在全球肆虐，導致各國邊境管制變嚴格，姊妹校的交換名額有所變動，以及學生本身出國意願低落的緣故。

[ 洞悉 ]

歐洲

疫情以前，中大學生對亞洲及歐洲的姊妹校最有興趣。 疫情爆發以後，學生赴歐洲的意願在110學年度(2021-2022)顯著降低，連帶影響中大整體申請交換人數。 2022以後，隨著歐洲疫情趨緩，各國逐漸開放邊境及制定寬鬆的防疫政策，使得申請歐洲交換的人數迅速回升至疫情前水平， 並反超亞洲地區，成為最受中大學生歡迎的交換地區。

亞洲

在亞洲地區，疫情的影響似乎在111學年度較顯著，這也許是因為疫情於亞洲地區流行的時間點較近期， 加上部分地區嚴格的檢疫政策，都成為潛在的交換阻力，因此申請人數一直到疫情趨緩的112年才有所回升。

北美洲

北美洲地區的姊妹校大多在美國境內，因其地理位置遙遠且盛行資本主義，交換生需支付高額學費及交通住宿費，因此中大學生對其申請意願較歐洲及亞洲低。 疫情影響時間點與歐洲類似，同樣在110學年度影響申請人數最顯著，並在疫情後有所反彈。

[ 製作方法說明 ]

國際處提供五年共五份申請人數pdf檔，由手動方式輸入各地區人數於data.js檔中進行加總，運用選單(manu)分別呈現不同地區申請人數隨時間的變化趨勢。

  • 資料來源： 中大國際處

專業領域與交換學生地區之關係

[ 背景簡介 ]

學生選擇交換地區時，往往受到姊妹校的申請科系限制影響。 另外，期望獲得的成長也是學生選擇地區時考量的因素，而這些都有跟自身擁有的專業背景相關。 透過國際處提供的各院學生歷年赴姐妹校情況資料，可繪製長條圖進行專業領域與交換地區之分析。

[ 洞悉 ]

文、客院

文院及客院的學生於課堂學習外語及文學領域，可在歐洲國家持續精進語言能力、拓展文學與文化視野。 尤其中央大學設有英文及法文系，因此法國姊妹校數量相當多，文院同學的交換風氣也相當盛行，因此大多選擇赴歐交換。

理、地、工、資電、生醫學院

針對理工科的同學，大多希望交換的同時可以進修相關科學技術，或是預先探索理想的碩士學程，歐洲地區擁有不少技術名校，可能是吸引理工學院同學前往的原因。 另外，部分領學術域具有地域性特色，如地科院同學可能會選擇環境較相似的亞洲地區學習進修。 亞洲也有不少名校開放特定科系名額給理工學院同學，這都可能是理工同學喜愛交換至亞洲地區的原因。

[ 製作方法說明 ]

國際處提供各院學生歷年赴姐妹校情況之ppt檔案，藉由手動方式將資料紀錄於data_b.js檔中並進行加總。

  • 資料來源： 中大國際處

各縣市住宅建材比例

[ 背景簡介 ]

各地常見的住宅建材與當地歷史發展相關，而建材種類會影響建物的耐震強度，進而導致不同程度的地震風險。 內政部不動產資訊平台提供不同年度各縣市戶籍及建材之統計資料，依此繪製圓餅圖進行建材比例分析。

[ 洞悉 ]

台北市

透過圓餅圖，可以看到台北市的鋼筋混凝土建物(RC)戶數比例非常高，可能原因為高度都市化發展導致戶口數量多而密集。 以RC建造之建物可以達15層樓以上，可以有效利用寶貴的都市土地，使RC成為台北市的主要建材。 值得一提的是，台北市也擁有一定比例的鋼骨混凝土(SS)及鋼骨鋼筋混凝土(SRC)建物，這兩種建材基於良好的彈性有利於強化建物耐震能力，因此常用於建造豪宅及摩天大樓。 台北市的SS與SRC比例高於全台平均，也暗示台北市擁有較多高消費力族群。

台南市

台南市眾所皆知的古都，擁有許多古蹟建築。圓餅圖中可看到約有62%的住宅使用現代鋼筋混凝土RC建造，與其都市化活動相關。 另外，有超過30%的住宅使用較傳統的磚造及加強磚造工法，應是基於台南悠久的都市發展歷史，保留了許多三合院民宅及傳統建物。 然而，磚造房屋意味著高屋齡，也暗示建物面對地震時，基於材料特性可能面臨較大倒塌風險。 未來可能會經類似美濃地震的事件，因此建物耐震在台南是一個值得關心的議題。

花蓮縣

從花蓮縣的圓餅圖可觀察到，其鋼筋混凝土RC的比例明顯低於全台平均， 主要原因可能是因為地處交通相對不便的東部，使花蓮縣的都市發展不如六都快速，因而保留許多傳統工法建物，如磚造、加強磚造等，其建物戶數比例(49.1%)甚至高於RC建物占比。 然而，花蓮屬於高地震危害區域，高比例的磚造建物對於強烈震動的耐震能力不佳，也因此花蓮縣可能需要更加重視房屋補強及定期結構評估等事宜。

[ 製作方法說明 ]

由內政部提供公開資料csv檔，挑選重點分析城市後輸入至data_p.js中，並進行圓餅圖比例分析。

  • 資料來源： 內政部不動產資訊平台
